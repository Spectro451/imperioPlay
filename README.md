<h1 align="center">ImperioPlay</h1>

**ImperioPlay** es un proyecto fullstack para una plataforma de e-commerce especializada en videojuegos. El objetivo principal es practicar nuevas tecnologías mientras se desarrolla un sistema completo y realista, listo para implementación en un entorno profesional.
El proyecto esta organizado como un monorepo en el cual en sus 2 secciones se explicara mas a detalle que incluye cada una

<details>
<summary>Funcionalidades Principales</summary>

### Vistas Cliente
- Catálogo interactivo de juegos con filtros avanzados
- Carrito de compras y lista de deseos
- Categorización y organización de productos por tipo/consola

### Gestión de Inventario
- Panel administrativo completo para productos
- Gestión de promociones y ofertas especiales

### Sistema de Intercambios
- Plataforma para intercambiar juegos
- Validación de transacciones y seguridad en intercambios
- Historial completo de intercambios realizados

### Ventas
- Proceso de compra completo con checkout optimizado
- Gestión de múltiples métodos de pago
- Generación automática de facturas y comprobantes

### Reportes y Análisis
- Reportes de ventas e intercambios
- Análisis de productos más populares y rentables
- Dashboard administrativo con métricas clave
</details>
